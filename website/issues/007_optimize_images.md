# Optimize Images

**Status:** Closed
**Priority:** Low
**Tags:** performance, assets

## Description
Ensure all images (logos, diagrams) are optimized. Use SVGs where possible and WebP for raster images. This ensures the fastest possible load times.

**Action Items:**
- [ ] Audit current image assets.
- [ ] Convert any PNG/JPGs to WebP or SVG.
- [ ] Ensure SVGs are minified.
