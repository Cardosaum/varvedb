# Migrate to Astro or VitePress

**Status:** Open
**Priority:** High
**Tags:** architecture, documentation, modernization

## Description
Migrate the website from vanilla Vite to a modern static site generator like **Astro** or **VitePress**. This is a strategic move to unify the marketing site and the documentation (currently in `mdbook`) into a single, cohesive developer portal.

**Benefits:**
- Unified design for docs and landing page.
- Better performance and SEO out of the box.
- Ability to use React/Vue components for interactive demos.
- Easier content management (Markdown/MDX).
