Institutional Grade Reliability

Zero-Copy Speed

Bypass serialization entirely. rkyv maps disk bytes directly to Rust types for instant access.

Crash-Safe Reliability

Built on LMDB for rock-solid ACID transactions. Your data survives crashes, power loss, and system failures.

Native Async

First-class tokio integration. Push-based change notifications let you build reactive systems that scale.

Military-Grade Encryption

Optional AES-256-GCM encryption with per-stream keys. Rigorous security for sensitive event logs.

Architecture

Designed for throughput and reliability. A clean separation of concerns.
