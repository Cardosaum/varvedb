Institutional Grade Reliability

Zero-Copy Persistence

Leverages rkyv to map events directly from disk to memory, eliminating serialization overhead and enabling sub-microsecond reads.

ACID Guarantees

Built on top of LMDB to provide full ACID transactions. Writes are atomic, consistent, isolated, and durable, ensuring data integrity.

Asynchronous & Reactive

First-class support for tokio. Subscribe to event streams in real-time with tokio::watch for building responsive architectures.

Cryptographic Security

Optional authenticated encryption using AES-256-GCM with AAD binding. Data is secured at rest with per-stream keys.

Architecture

Designed for throughput and reliability. A clean separation of concerns.
